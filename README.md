# 📊 RKDash — Аналитическая система для Rkeeper

**RKDash** — это современная аналитическая платформа для ресторанов на базе **Rkeeper**. Система позволяет визуализировать ключевые метрики бизнеса, общаться с данными через AI-агента, отслеживать план/факт, контролировать премии и штрафы, а также выявлять подозрительные операции.

🔗 **Демо:** [rkdash.geniok.ru](https://rkdash.geniok.ru)

---

## 🚀 Возможности

### 📈 Панель управления
- Общая аналитика по продажам, среднему чеку, количеству гостей
- Визуализация динамики выручки (графики, диаграммы)
- Фильтрация по датам и заведениям

### 🤖 Чат с базой данных (AI)
- Задавайте вопросы на естественном языке
- AI-агент анализирует данные Rkeeper и отвечает в реальном времени
- Пример: *«Какая выручка была вчера?», «Топ-5 блюд за неделю»*

### 📋 План/Факт
- Сравнение плановых показателей с фактическими
- Визуальное отображение отклонений
- Контроль выполнения KPI

### 🏆 Премии и штрафы
- Автоматический расчёт мотивации персонала
- Правила на основе данных Rkeeper
- Прозрачная система начисления

### ⚠️ Подозрительные операции
- Выявление аномалий в продажах
- Контроль возвратов, списаний, скидок
- Помощь в предотвращении фрода

### 🔮 Прогнозирование
- AI-прогнозы продаж на основе исторических данных
- Планирование закупок и загрузки персонала

## 🛠️ Стек технологий

| Компонент | Технология |
|---|---|
| **Фреймворк** | Next.js |
| **Язык** | TypeScript |
| **UI** | shadcn/ui, Radix UI, Tailwind CSS |
| **Иконки** | Tabler Icons |
| **Графики** | Recharts |
| **AI** | OpenAI API / Агентный подход |
| **База данных** | PostgreSQL |
| **ORM** | Drizzle ORM |

---

## 🎯 Для кого этот проект

- **Владельцы ресторанов** — прозрачная аналитика бизнеса
- **Управляющие** — контроль план/факт, мотивация персонала
- **Шеф-повара** — анализ популярности блюд, списаний
- **Системные администраторы Rkeeper** — расширение возможностей учётной системы
